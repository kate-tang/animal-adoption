# animal-adoption

> A Vue application for people looking to adopt animals.

## Get Started

``` bash
# Install dependencies
npm install
# Compiles and hot-reloads for development
npm run serve
# Compiles and minifies for production
npm run build
```

## Tools and Technologies

* Vue 3
* SCSS
